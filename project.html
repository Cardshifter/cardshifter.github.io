The Cardshifter Project

Cardshifter is an open-source Trading Card Game (TCG) project which began in August 2014. It is being developed by a team of dedicated software programmers, web developers and designers. This page will go over some of the tools and technologies that we employ.

Git, Github

The Cardshifter Github repository is the central hub where all the code and files are stored. It is a vastly valuable tool in facilitating teamwork, version control, and just generally keeping things organized. Should you decide to become a contributor to the Cardshifter project as a programmer, this is where your journey will begin.

Bootstrap CSS

This website is built in part using the Bootstrap CSS library, which makes it easy and fast to make professional-looking websites. Special thanks to MaxCDN for hosting the Bootstrap CSS library used for this site. If you wish to use it for your site, you may add the following into the <head> of your website:

PostgreSQL

Cardshifter will be using a normalized PostgreSQL relational database in order to gather a plethora of game statistics which will be displayed both in-game and available on the web. That way, players can keep track of their performance in each mod, and with each version of their decks, to see how they rank against other players.

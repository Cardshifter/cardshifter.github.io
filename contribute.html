Contribute

The Cardshifter team is always open to welcoming new talent into the project! The following are some ways you can get involved.

Try the game!

This is a great way to familiarize yourself with the client, the gameplay and the features, while having a good time. You can download the latest release absolutely free and connect to one of our servers to play with others, or run a local instance and play against one of our Artificial Intelligence players. The server information is below.

IP Address: dwarftowers.com | Port: 4242

Visit Github

Visit the Cardshifter repository on Github and Star it, Watch it and even become a Contributor! There is a good amount of information on the Github Wiki to help you get started. You can also view currently open tasks on the Open Issues page.

Join the chat room

We have a chat room on the Stack Exchange network dedicated to this project, where we discuss day-to-day things and also hang out for fun. There is also a feed which keeps us informed of Github activity.

Please note that you will need at least 20 reputation points on the Stack Exchange network in order to be able to speak. You can view the discussion without any reputation, though. You can easily get 20 reputation by posting a question or an answer on any of the stack Exchange sites, including Code Review (where the project initiated) and Stack Overflow.

Contact us

You may use the links on the left navigation bar in order to contact us, report a bug or request a feature. You may also report a bug or request a feature by opening a new issue on Github Issues page.

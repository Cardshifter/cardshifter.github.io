Play Cardshifter!

The Cardshifter team strives to make the game not only fun and interesting, but also easy to use and compatible on many platforms. This page will guide you through a few simple steps to set up the game and start playing Cardshifter today!

Java version

You will need Java 8 or above to be able to play Cardshifter. Java 8 by Oracle is completely free and works on all modern operating systems.

Java 8 is the official Java version since October 2014. If you are unsure of your Java version, you can visit this link to find out. If you do not have Java 8, you can download it for free from the same website.

Get the latest release

You will first need to download the latest release in order to play Cardshifter. Once you have saved it to your computer, you can either open the terminal and run java -jar cardshifter-fx-<version>.jar or you can simply double-click the cardshifter-fx-<version>.jar file.

Connect and play!

There is a server running at dwarftowers.com | port 4242 that you may connect to and play with other players as well as AI players. Just choose your user name and connect to enter the lobby. If this is your first time playing Cardshifter, you may want to read the Game Rules.

Once you are in the lobby, you can select the Deck Builder to build and save your own deck, or just invite a player or AI to a game, which will bring up the Deck builder where you can load an existing deck or create a new one. Then just select Start Game and enjoy!

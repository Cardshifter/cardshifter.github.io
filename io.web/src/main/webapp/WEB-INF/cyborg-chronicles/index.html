Cyborg Chronicles

Cyborg Chronicles is the first mod (short for "modification") using the Cardshifter trading card game platform. It is a dystopian, cyberpunk themed game featuring human factions, war machines and interplanetary conflict. It can be played online absolutely free and is compatible with all major operating systems and soon with mobile platforms, so you can challenge your friends no matter what device your prefer!

Browser-based web client almost ready!

Published: 2015-08-24

Summary

We have been very busy last 3 weeks developing a brand new browser-based, fully functional game client. This means players will be able to connect and play instantly. No downloads, no installs, no updates, just type your name and connect!

Follow @Cardshifter on Twitter for updates!

Fun facts about web client:

  • Combining the best of web app technologies for a clean and responsive design.
  • For the first time, card images and animations will be here!
  • Flexible, supporting effect and flavor text, and visual card metrics using Bootstrap CSS.
  • Will allow custom CSS templates for each mod, thanks to some AngularJS magic.
  • Server-side game data passed via WebSocket ensures that all client instances are playing the same version of the game, with nothing at all for the player to download or install.
  • Continuing development with the goal of flexibility and no-nonsense design.
  • Tested in Firefox, Google Chrome, Safari, and Internet Explorer 9 or newer. (no plans to support IE 8 or older)

Published by Cardshifter dev team
